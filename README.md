# project
Часть проекта, отвечающая за Web-клиент. 
Web Client — модуль системы массового тестирования и опросов.

Модуль отвечает за:
- отображение пользовательского интерфейса в браузере;
- взаимодействие пользователя с системой;
- перенаправление пользователя на авторизацию;
- отображение данных, полученных от серверной части приложения.

Интерфейс реализуется с использованием HTML, CSS и JavaScript.
